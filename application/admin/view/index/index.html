后台介绍

于2017年5月9日竣工完毕，历时3个月完成，后台采用bootstrap+thinkphp形式，账号分为主账号和游客账号两个，游客账号权限仅限于浏览无法进行操作。如果有兴趣进行交流或者发现了什么BUG请发送邮件至1845348325@qq.com。写给自己的注意事项：

  1. 今日新增为当天的文章数量。
  2. 总浏览数为博客网站浏览量。
  3. 横向幻灯片高度250宽度建议1920。
  4. 竖向幻灯片高度建议超过1200宽度300。
  5. 头像上传的文件建议为jpg,虽然其他格式的图片也会被强制转换成jpg。
  6. 网站文章编辑器使用的是百度的ueditor，音乐功能测试过后发现BUG没办法修复。
  7. 备份还是很重要的一周备份一次吧。
